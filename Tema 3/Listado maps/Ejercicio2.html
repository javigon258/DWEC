2. Tanto los arrays javaScript como los Sets permiten almacenar elementos. Indica la diferencia entre ambos.

La diferencia entre arrays y set son:

  • Array, contiene cualquier valor que se añada, y con Set, contiene solo valores únicos.
  • Array, Enfocado a indexar. Necesitas escribir código para encontrar y borrar un elemento en particular, y con Set, enfocado al valor. Puedes eliminar un valor fácilmente.
  • Array, .push() para agregar un elemento, y con Set, .add() para agregar un elemento.
  • Array, .includes() (ES7) para encontrar si un elemento está allí en una matriz, y con Set, .has() para encontrar si hay un elemento en un conjunto.
  • Array, .length para encontrar cuantos elementos, y con Set, .size encontrar numero de elementos.
  • Array, arr.splice(0,arr.length) para vaciar una matriz, y con Set, mySet.clear() para borrar un conjunto.
