Tema 3: Objetos predefinidos

  1. Añade elementos a la lista existente. Almacénalos en un set, en caso de repetirse alguno, lanza una excepción y captúrala.
  2. Almacena en un array fechas mediante algún constructor de Date y muestra por pantalla las fechas de los años bisiestos.
  3. Pasa a JSON el objeto {nombre: "Mario", apellidos: "Navarro Madrid", aprobado: false}. Muestralo y vuelve a convertir el JSON en objeto. Muestra sus atributos.
  4. Crea un formulario donde se deba insertar un número de teléfono, un usuario y una contraseña. Se deben validar mediante una expresión regular. El usuario debe contener cualquier carácter y mínimo un dígito y la contraseña debe contener al menos una mayúscula, una miníscula, un dígito y un caráctere especial.
  5. Mediante un objeto Number y algún método de Math, expón un númerodado a través de un formulario hasta 10.
