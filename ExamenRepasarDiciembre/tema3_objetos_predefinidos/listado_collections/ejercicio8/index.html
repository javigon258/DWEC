Responde con respecto a WeakSet:

  1. Un conjunto de elementos de tipo: objetos
  2. Constructor admite como parámetros: un iterable
  3. Métodos para añadir: add(value)
  4. Métodos para eliminar: delete(value)
  5. Métodos para buscar: has(value)
  6. Número de elementos: No existe ningún método para ello
  7. Si dos elementos son iguales: Si la referencia de los elementos son iguales, no se añade
  8. Se recorren mediante: no existe ningún método para ello
