Responde con respecto a Set:

  1. Un conjunto de elementos de tipo: valores de cualquier tipo
  2. Constructor admite como parámetros: un iterable
  3. Métodos para añadir: add(value)
  4. Métodos para eliminar: delete(value) y clear()
  5. Métodos para buscar: has(value)
  6. Número de elementos: size
  7. Si dos elementos son iguales: se reemplaza el que insertas por el que está en el set
  8. Se recorren mediante: foreach y for of
